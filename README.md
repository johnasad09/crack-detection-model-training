# Concrete Crack Detection

This repository contains three machine learning models trained on two distinct datasets for detecting concrete cracks.

---

## 📂 Models Included

1. **Logistic Regression Model** (`concrete_crack_detection_logistic_regression.ipynb`)
   * Performs binary classification to label images as **"Crack"** or **"No Crack"**.
2. **YOLO (Bounding Box Object Detection)** (`concrete_crack_detection_yolo_bounding_box.ipynb`)
   * Identifies and draws bounding boxes around detected cracks.
3. **YOLO (Semantic Segmentation)** (`concrete_crack_detection_semantic_segmentation.ipynb`)
   * Performs pixel-level segmentation to outline the exact shape of cracks.
   * Github cannot rendered the file so we included a pdf file as well.
